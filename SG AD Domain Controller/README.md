# Template for Service Group "SG AD Domain Controller"
Contains following Services:
* DNS (53/TCP/UDP)
* LDAP (389/TCP/UDP)
* LDAP SSL (636/TCP)
* LDAP GC (3268/TCP)
* LDAP GC SSL (3269/TCP)
* NTP (123/UDP)
* RPC Endpoint Mapper (135/TCP)
* Kerberos (88/TCP/UDP)
* Kerberos password change (464/TCP/UDP)
* Windows RPC (49152-65535/TCP)
* SMB (445/TCP)
